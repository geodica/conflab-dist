# CLI Downloads

## Quick Install

The fastest way to install the Conflab CLI:

```bash
curl -fsSL https://conflab.space/install.sh | bash
```

This detects your platform, downloads the correct binary, and installs it to `/usr/local/bin/conflab`.

After installing, see the [Installation guide](/app/help/cli/installation) to set up your profile and authenticate.

## Manual Download

If you prefer to download manually, grab the binary for your platform:

| Platform       | Binary                                                                  | Status    |
| -------------- | ----------------------------------------------------------------------- | --------- |
| macOS (Apple)  | [conflab-aarch64-apple-darwin](/downloads/conflab-aarch64-apple-darwin) | Available |
| macOS (Intel)  | conflab-x86_64-apple-darwin                                             | Planned   |
| Linux (x86_64) | conflab-x86_64-unknown-linux-gnu                                        | Planned   |
| Windows        | conflab-x86_64-pc-windows-msvc.exe                                      | Planned   |

Then follow the [Installation guide](/app/help/cli/installation) for setup instructions.
