# Favourites

**Favourites** is your personal list of Catalog entries that you return to often. It is a stronger signal than Library: Library is "I might want this later", Favourites is "I want this at my fingertips".

## Marking an Entry as a Favourite

From any Catalog entry detail page at `/app/lsd/entries/<slug>`:

1. Click the **Favourite** button (heart icon) on the action bar.
2. The icon fills in to indicate the entry is now a favourite.

Click again to unfavourite.

## Viewing Your Favourites

Open `/app/lsd` and switch to the **Favourites** tab. You see every entry you have favourited, sorted by most recently favourited.

The Favourites tab is only visible to signed-in users.

## Favourites vs Library

Both are personal collections, but they serve different purposes:

| Library                      | Favourites                      |
| ---------------------------- | ------------------------------- |
| "I might want this later."   | "I want this at my fingertips." |
| Broader reading list.        | Curated must-haves.             |
| Larger collection over time. | Stays small and relevant.       |
| Save button on entry detail. | Favourite (heart) button.       |

You can have an entry in both Library and Favourites. Unfavouriting does not remove from Library.

## Privacy

Favourites are private. Other users cannot see which entries you have favourited. Favourite counts on entries are not exposed.
